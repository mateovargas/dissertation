Xeno-Plague: A Discrete SIR Game for Understanding Epidemics

Developed by Mateo Vargas

Music by Kyng Cold

Developed using Unity IDE version 2017.1.0f3 Personal (64bit)

To view the source files for the simple SIR script, navigate to ../Simple_SIR_Script

Please use Unity to view source files sound in ../Dissertation/ . Unity IDE can be downloaded here here: https://unity3d.com/

To run a prototype build of the game, navigate to ../Builds/ and select the build for your preferred operating system. Note that this game was specifically tested and deployed Windows 10 and OSX 10.12.6.
